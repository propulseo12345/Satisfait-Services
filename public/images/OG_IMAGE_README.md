# Image Open Graph (og-image.jpg)

Pour un partage optimal sur les réseaux sociaux, créez une image Open Graph de **1200x630 pixels**.

## Spécifications
- **Dimensions** : 1200 x 630 pixels
- **Format** : JPG ou PNG
- **Taille** : < 1 MB (recommandé)
- **Nom du fichier** : `og-image.jpg`

## Contenu recommandé
L'image doit contenir :
- Le logo Satisfait Services
- Un texte accrocheur : "Ménage & Nettoyage à Pau"
- Une image de fond représentant un intérieur propre
- Les couleurs de la marque (bleu primaire)

## Outils pour créer l'image
- Canva (modèle "Open Graph Image")
- Figma
- Photoshop
- Outils en ligne : og-image.vercel.app

Une fois créée, placez l'image dans `/public/images/og-image.jpg`
